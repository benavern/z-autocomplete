z-autocomplete demo

Get started

z-autocomplete has been design as a result of an attempt to using the github auto-complete element but didn't fit my needs. It is highly inspired by it.

Install via NPM

The component is available here on npm, you can install it via this command:

npm i @benavern/z-autocomplete

Then you can use it in your scripts like so in your js/ts module files: 

import '@benavern/z-autocomplete';

Install via CDN

<script type="module" src="https://unpkg.com/@benavern/z-autocomplete"></script>

Have fun
